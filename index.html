Principales etiquetas HTML
Etiqueta Descripción
DOCTYPE Informa al navegador la versión de HTML usada
html Contiene el bloque de código que corresponde al documento
head Parte superior del documento donde se detallan los metadatos
meta Metadatos del documento, los cuales aportan información sobre este, como la descripción, autor, palabras clave, entre otros
tittle Título que se visualiza en la barra superior del navegador
body Contenido del documento HTML
h1...h6 Niveles de títulos que permiten estructurar el documento
div Permite estructurar el documento en secciones
a Direcciona a un enlace
strong Permite resaltar una parte del texto en negrita
br Genera un salto de línea
img Permite insertar imagenes al documento
p Agrupa texto dentro de un parágrafo

Estilos que pueden aplicarse en CSS
Estilo Descripción Sintaxis
Colors Define el color del elemento color: [color]
Backgrounds Define el color de fondo del elemento background-color: [color]
Borders Define las propiedades de los bordes border: [ancho | estilo | color]
Margins Establece el margen margin: [margen]
Fonts Establece las propiedades de la fuentes de un texto font: [fuente]
Height Especifica la altura de un elemento height: [altura]
Width Especifica el ancho de un elemento width: [ancho]
Position Determina la ubicación de un elemento dentro del documento position: [posicion]
Overflow Especifica qué se debe mostrar cuando el contenido se derborda overflow: [overflow]
Float Ubica el elemento a uno de los lados del contenedor float: [ubicación]
